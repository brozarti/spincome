# Spincome Advertiser Pilot — Outreach Kit

**Goal:** get 2+ paid pilot campaigns ($200 each) from real dev-tool companies within 2 weeks.
This is the demand test that decides whether Spincome deserves more investment.
**Success = 2+ pilots close. Failure = 0 pilots after 15 sends + 1 follow-up each → stop building, pivot.**

## The pilot offer (already live on spincome.io/advertise)

- $200 budget, locked $8 CPM → ~25,000 impressions
- Full transparency: impression logs with tool context, timestamps, unique developer counts, clicks/CTR
- Unspent budget refunded, no questions asked
- Self-serve: they fill the form, pay by card via Stripe, campaign is live in minutes

**Why $8 CPM is credible:** developer newsletter sponsorships run $60–150 CPM
([SponsorGap 2026 guide](https://sponsorgap.com/blog/newsletter-sponsorship-rates-2026)), niche dev
newsletters $25–150 ([Wellput](https://www.wellput.io/blog/technology-newsletters-worth-advertising-in)),
dev podcasts $25–65 ([daily.dev](https://business.daily.dev/resources/developer-podcast-advertising-sponsorship-formats-costs-roi/)).
We undercut the cheapest comparable channel ~3x, because we're new and our network is small. Say that plainly — honesty converts better with dev-tool marketers than hype.

## Honesty rules (do not break these — the product survives on trust)

- Never claim audience size we don't have. Current network is small; the pitch is "founding rate + full logs," not reach.
- Report real numbers only: impressions, unique developers, clicks. No "estimated views."
- Acknowledge the incentive question if asked: developers are paid to receive ads, so we sell
  *brand awareness to Claude Code power users*, not conversion intent. Cheap CPM prices that in.
