# Promo Carousel Playbook — reusable for every future project

Distilled from viral dev-content carousels, rebuilt to fit an honest product. Use this to make
screenshot-ready promo slides for X / LinkedIn / Reddit. Live template artifact for Spincome:
the 6-card terminal carousel (💸) — reuse its structure, swap the content.

## The 6-card skeleton (works for any product)

1. **Hook** — the single most characteristic thing about the product, shown not told. For a
   terminal product, render an actual terminal. For a visual product, show the output. Headline is
   a benefit in ≤5 words ("Get paid while you code").
2. **The payoff** — one number or one outcome, big. What the user walks away with.
3. **Trust / objection-killer** — name the thing people will worry about and disarm it ("It never
   reads your code"). Concrete, not reassuring fluff.
4. **How it works** — 3–4 numbered steps. Only number things that are genuinely sequential.
5. **The other side** — for a marketplace, the second audience (advertisers). For a solo product,
   proof or a use-case.
6. **CTA** — one command / one URL. Nothing else competing for the click.

## Design rules (what made the rebuild better than the source)

- **Earn your accent from the subject.** The source decks all used the same acid-green-on-black —
  that reads as AI-generated. Spincome's carousel uses the product's *real* emerald and an actual
  terminal aesthetic because it literally renders ANSI ads. Derive the look from what the thing IS.
- **Copy is honest or it dies on Reddit/HN.** Every claim must be literally true. The last card
  says "coffee money, not a paycheck" on purpose — undersell, then overdeliver. No refund promises,
  no "advertisers love it," no invented numbers. (See [[spincome-outreach-rules]].)
- **One message per card.** If a card needs two sentences to explain, it's two cards.
- **Screenshot-first.** Fixed card size, dark ground, big type — each card must stand alone when
  someone screenshots just that one and reposts it.
- **Motion is seasoning.** One blinking cursor, one count-up. Respect `prefers-reduced-motion`.

## Posting

Space platforms out (don't blast the same link everywhere within an hour — reads as spam and can
flag the domain). HN/Reddit first with the honest framing, then X threads pulling individual cards.
