# Reusable Prompt Template — the honest skeleton

Extracted from the "SMB" mega-prompt carousel (Superhuman Learning / God-Tier Life / Think Like a
Billionaire, etc.). Those prompts work *mechanically* but drown the mechanics in hype. Here's the
skeleton with the theater removed — use it to brief an AI on any real skill or goal.

## The structure that actually does the work

    Role:        "Act as a [specific expert] who has taken beginners to [concrete outcome]."
    Goal:        "I want to [specific, measurable goal] by [honest timeframe]."
    Method:      "Use [named techniques]: spaced repetition, active recall, interleaving,
                  the Feynman technique." (Name real methods — this is the load-bearing part.)
    Staging:     "Break it into stages, each with tasks, resources, and a practice assignment
                  I can actually do."
    Feedback:    "After each stage, quiz me and correct what I got wrong."
    Constraints: "Assume [my starting level / time budget / tools]. Flag anything unrealistic
                  instead of flattering me."

## Why the hype versions underperform

- "Top 1% in 90 days," "god-tier," "leapfrog everyone" set false expectations and make the model
  generate motivational filler instead of a usable plan. Cut them.
- The real leverage is: **named learning techniques + staged progression + active practice +
  honest constraints.** That's it. Everything else was decoration.
- Ask the model to *push back on unrealistic goals* — the single most useful line, and the exact
  opposite of what the hype prompts do.

## Fold this into future project briefs

When kicking off any learning- or research-heavy task, start from this skeleton rather than a vague
ask. It's the same discipline as the [[promo-carousel-playbook]]: keep the working parts, drop the
performance.
